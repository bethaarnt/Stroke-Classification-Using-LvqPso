# Stroke Classification App Using LvqPso

### Cara Install Aplikasi

Sebelum memulai langkah-langkah berikut, pastikan sudah menginstall library python

1. Unduh file aplikasi dalam format ZIP atau lakukan cloning repository.
2. Ekstrak file ZIP, lalu arahkan ke direktori aplikasi yang telah diekstrak.
3. Buka command prompt (cmd) atau Git Bash dan jalankan perintah-perintah berikut secara berurutan.  
4. Buat folder virtual envirorment: **python -m venv env** , lalu masuk ke dalam folder tersebut: **cd env**
5. Install requirement aplikasi: **pip install -r requirements.txt**
6. Keluar dari direktori virtual envirorment: **cd ..**
7. Pindah direktori ke TugasAkhir untuk menjalankan aplikasi: **cd TugasAkhir**
8. Jalankan aplikasi: **python manage.py runserver**
9. Setelah server berjalan, salin URL server ke browser: http://127.0.0.1:8000/
10. Untuk menghentikan server, cukup tekan Ctrl + C pada cmd/terminal yang digunakan untuk menjalankan server.
